## 使用说明

1. **下载文件**  
   您可以通过本仓库下载所需的 RPM 文件包。

2. **安装 RPM 文件**  
   下载完成后，使用以下命令安装 RPM 文件：
   ```bash
   sudo rpm -ivh <文件名>.rpm
   ```
   例如：
   ```bash
   sudo rpm -ivh cvuqdisk-1.0.10-1.rpm
   ```

3. **验证安装**  
   安装完成后，您可以使用以下命令验证 RPM 文件是否成功安装：
   ```bash
   rpm -qa | grep <文件名>
   ```
